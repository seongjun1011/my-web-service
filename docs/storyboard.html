1. 서비스 개요

항목내용
서비스명SmartPantry
목적AI를 활용한 식재료 촬영/영수증 인식으로 냉장고 식재료를 자동 등록·관리하고, 유통기한 임박 알림 및 레시피 추천을 제공하는 스마트 저장고 관리 서비스
대상 사용자식재료 관리가 번거로운 일반 가정 사용자
플랫폼모바일 웹 (PWA), Chrome/Safari 지원
로그인 방식카카오 로그인, Google 로그인 (소셜 OAuth)
핵심 기능AI 식재료 인식, 영수증 OCR, 유통기한 관리, 레시피 추천, 푸시 알림

3. 화면별 스토리보드

3.1 로그인 화면

LOGIN
구성 요소설명
앱 로고쇼핑바구니 아이콘 + "SmartPantry" 텍스트
카카오 로그인 버튼노란 배경, 카카오 공식 색상. 클릭 시 카카오 OAuth 페이지로 이동
Google 로그인 버튼흰색 배경, Google 공식 디자인. 클릭 시 Google OAuth 페이지로 이동
로그인 성공 시: 기존 회원 → 홈 화면 / 신규 회원 → 약관 동의 화면

3.2 약관 동의 화면

TERMS
구성 요소설명
제목"서비스 이용 약관 동의"
약관 내용 영역스크롤 가능한 개인정보 수집·이용 동의 텍스트
동의 버튼"모두 동의하고 시작하기" → DB에 is_agreed=1 저장 후 홈 이동
최초 1회만 표시. 동의 후 재방문 시 바로 홈 화면

3.3 홈 화면

HOME
구성 요소설명조건
공지 배너관리자가 등록한 공지 메시지. X 버튼으로 닫기 가능활성 공지 있을 때만
인사말"안녕하세요, {닉네임}님! 👋"항상
유통기한 임박 섹션D-3 이내 식재료를 가로 스크롤 카드로 표시. D-Day/경과 일수 색상 강조임박 재료 있을 때
신선 메시지"✅ 모든 재료가 신선해요!" 초록 배경임박 재료 없을 때
인식 모드 선택식재료 촬영 / 영수증 스캔 탭 전환항상
스캔 버튼선택된 모드에 따라 색상 변경 (식재료: 검정, 영수증: 파랑)항상
오늘의 추천 레시피AI 추천 버튼 → 2개 썸네일 카드 표시. 클릭 시 레시피 탭 이동항상

3.4 내 냉장고 화면

PANTRY
구성 요소설명
헤더"내 냉장고" 제목 / 삭제 모드 토글 버튼(-) / 추가 버튼(+)
카테고리 탭전체 / 냉장 / 냉동 / 실온 필터링
식재료 목록이모지 + 이름 + 카테고리 뱃지 + D-Day 스펙트럼 (빨강→초록, 14일 기준)
삭제 모드- 버튼 클릭 시 체크박스 등장 → 항목 선택 → "N개 삭제" 버튼 표시
수동 등록 모달품목명 / 소비기한 / 보관장소 입력 후 저장
빈 상태"저장고가 비어있습니다" 안내 메시지

D-Day 색상 기준

남은 일수색상
0일 이하 (경과)빨강 (D+N)
1~3일주황
4~7일노랑
8~14일초록

3.5 식재료 스캔 화면

CAMERA
구성 요소설명
모드 안내 배지상단 중앙: "🥦 식재료를 카메라에 맞춰주세요" / "🧾 영수증을 펼쳐서 촬영해주세요"
카메라 뷰후면 카메라 전체 화면 라이브 뷰
촬영 버튼하단 중앙 흰색 원형 버튼 → 클릭 시 캡처 후 스캔 결과 화면 이동
닫기 버튼좌측 상단 ← 버튼 → 카메라 스트림 종료

3.6 스캔 결과 화면

SCAN RESULT
구성 요소설명
촬영 이미지정사각형 카드로 촬영된 이미지 표시
인식 중 상태스피너 + "🥦 식재료 AI 인식 중..." / "🧾 영수증 OCR 인식 중..." (최대 2분 소요 안내)
인식 결과 목록체크박스 + 이모지 + 식재료명 + 유통기한 + 보관방법 뱃지
항목 선택/해제개별 항목 클릭으로 저장 여부 선택
저장 버튼"선택 항목 저장고에 추가" → 냉장고에 일괄 등록
오류 상태에러 메시지 + "다시 시도" / "다시 촬영하기" 버튼

AI 인식 방식

모드처리 방식결과
식재료 촬영YOLO 모델(smai_v1.pt)로 식재료 감지감지된 식재료 + 자동 유통기한 계산
영수증 스캔LLM 1패스(OCR) → 2패스(정규화)상품명 정제 + 구매일 기반 유통기한 계산

3.7 레시피 추천 화면

RECIPE
구성 요소설명
헤더"오늘의 추천 🍽️" + 새로고침 버튼 (추천 후 표시)
재료 칩 목록pantry에서 가져온 재료 표시. 임박 재료는 빨간 칩, 일반은 회색 칩
추천 버튼"🤖 보유 재료로 레시피 추천받기" → LLM API 호출
로딩 상태스켈레톤 카드 2개 애니메이션 + "AI가 레시피를 생각하는 중..." 안내
레시피 카드그라디언트 배너 + 음식 이모지 / 요리명 / 시간 / 난이도 / 사용·부족 재료
조리 방법"조리 방법 보기" 토글 → 단계별 번호 리스트 펼치기
임박 재료가 앞쪽에 정렬되어 LLM에 전달됨 → 임박 재료 우선 활용 레시피 추천

3.8 마이페이지

MY PAGE
구성 요소설명
프로필 섹션사용자 아이콘 + "{닉네임}님" 표시
알림 켜기/끄기Web Push 구독 토글. 켜짐 상태 뱃지 표시
개인정보 설정바텀시트 → 아코디언으로 3가지 메뉴 펼치기
ㄴ 닉네임 변경텍스트 입력 → 저장 → 화면 이름 즉시 반영
ㄴ 데이터 초기화전체 식재료 삭제 (확인 후 실행)
ㄴ 회원 탈퇴계정 + 모든 데이터 영구 삭제 → 로그인 화면
고객센터카카오 채널 채팅 페이지로 이동
로그아웃세션 종료 → 로그인 화면

3.9 관리자 대시보드

ADMIN (관리자 계정 전용)

일반 사용자에게는 보이지 않음. DB에서 is_admin=1인 계정만 접근 가능.

탭구성 요소설명
📊 통계 요약 카드 전체 유저 수 / 전체 식재료 수 / 유통기한 임박(D-3) 수
소셜 로그인 비율 카카오/구글 가입자 비율 바 차트
최근 7일 가입자 날짜별 신규 가입자 수 인라인 차트
👥 유저 관리 유저 목록 이름 / 소셜 제공자 / 식재료 수 표시
식재료 펼치기 유저 카드 클릭 시 해당 유저의 식재료 목록 표시
강제 탈퇴 휴지통 버튼 → 확인 후 해당 유저 계정·데이터 삭제
📢 공지 관리 공지 등록 텍스트 입력 후 등록 → 전체 유저 홈화면에 배너로 표시
활성/비활성 토글 토글 스위치로 공지 켜고 끄기
로그아웃 헤더 우측 버튼 → 세션 종료 후 로그인 화면
